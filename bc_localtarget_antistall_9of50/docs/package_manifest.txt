DESCRIPTION
-----------
Target-conditioned Behavior Cloning (BC) policy for Franka 5-cube stacking.
Uses RNN policy, LocalTarget representation, anti-stall guard, planner-forced
discrete gripper, and 7D action space (dx, dy, dz, rx, ry, rz, gripper).

This is a Stage A/B prototype:
  Stage A: HDF5 demos from scripted StackPlanner / expert controller
  Stage B: target-conditioned BC RNN with LocalTarget + anti-stall guard
  Stage C: RL fine-tuning — NOT completed
